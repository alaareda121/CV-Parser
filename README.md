# CV Parser using LLMs

This project parses CVs in PDF format and extracts structured information
using a Large Language Model (LLM) and schema-guided JSON extraction.

## Features
- PDF text extraction
- Schema-based CV parsing
- LLM-powered information extraction
- Streamlit GUI interface

## Notes
- Running the Streamlit app requires downloading a large language model.
- The model files are not included in this repository.

## How to run
```bash
streamlit run cv_parser.py
